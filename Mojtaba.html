HTML

Strukturen för en fungerande webbplats

Vad är HTML?

HTML står för HyperText Markup Language och är ett märkspråk, inte ett programmeringsspråk i traditionell mening. Det används för att strukturera och organisera innehållet på en webbsida. Med HTML kan man bestämma vad som ska visas på sidan, i vilken ordning, och hur innehållet är uppdelat i olika sektioner. HTML använder sig av element ("taggar") som omger olika delar av innehållet, till exempel rubriker, stycken, bilder eller länkar. Dessa element talar om för webbläsaren hur innehållet ska tolkas och visas.

Vad använder HTML?

HTML använder sig av flera olika element som bygger själva webbsidan. Normalt skriver man koden i en programmeringsmiljö som Visual Studio Code, men det finns många andra alternativ. Dessa verktyg hjälper utvecklare att skriva, färgmarkera och strukturera HTML på ett enklare sätt.

Code

<!DOCTYPE html>
<html lang="sv">
<head>
    <title></title>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="container">
        <h1>Hello World</h1>
        <p>and goodbye!</p>
    </div>
</body>
</html>

Det här är ett exempel på hur en grundläggande HTML-kodruta ser ut. Självklart finns det mycket mer kod i riktiga projekt, men detta visar de viktigaste elementen: h1 för rubriker, och div för avdelningar eller "boxar" som används för att organisera layouten på sidan.
