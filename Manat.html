Vad är CSS?

CSS, som står för Cascading Style Sheets, är en av de viktigaste delarna i en webbsida tillsammans med HTML och JavaScript. Med CSS kan du exakt styra hur en webbsida ska se ut och upplevas. Allt från färger och typsnitt till layout, visuella effekter och animations. Det ger dig möjlighet att skapa nästan vilken design du kan föreställa dig. CSS används för att bestämma webbsidans utseende och gör det enkelt att separera innehåll från design.

CSS tar den redan existerande strukturen på en webbsida och ger olika egenskaper, som position, storlek och färg. Till exempel, om du trycker på text rutan på font-size kan du ändra textens storlek, och om du klickar på färgrutorna vid background-color eller color kan du enkelt ändra färgen på webbplatsens bakgrund eller text.
